Descomplicando o Pacote Office

O Microsoft Word é um processador de texto, ele permite que os usuários criem e editem documentos de texto com facilidade, incluindo cartas, currículos, relatórios e até mesmo livros inteiros.

ㅤ+ Dicas e truques:

ㅤO software possui uma função bem interessante para ajudar você a não cometer erros ortográficos. Para isso, vá até a aba Arquivo, clique em Opções, escolha o ícone Revisão de Texto e altere as Opções de autocorreção de acordo com as suas necessidades - é possível optar entre nenhuma, manual e automática.


ㅤ+ Atalhos que facilitam:

Ctrl + O Cria um novo arquivo sem fechar o atual
Ctrl + Alt + V Ativa o colar especial (permite colar texto sem algumas características)
Ctrl + P Abre a página de impressão
Ctrl + T Seleciona todo texto
Ctrl + B Salva o arquivo
Ctrl + ENTER Quebra a página
Ctrl + Shift + A Formata todas as letras para maíusculas
Ctrl + F Abre uma guia de busca
Ctrl + Z Desfaz a última ação
Ctrl + Y Refaz a última ação
