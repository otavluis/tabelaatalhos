Descomplicando o Pacote Office

ㅤ ㅤ

O Excel é um programa de software que permite criar tabelas, calcular e analisar dados. Este tipo de software é denominado software de folha de cálculo. O Excel permite criar tabelas que calculam automaticamente os totais de valores numéricos introduzidos, imprimir tabelas em esquemas atrativos e criar gráficos simples.

ㅤ+ Dicas e truques:

ㅤEm muitos casos, onde é necessário copiar o argumento entre células, é necessário fixar tal argumento. Para fixar uma célula, basta inserir o símbolo $ antes da letra da coluna e do número da linha. No entanto, existe um jeito mais fácil: basta selecionar a célula e pressionar a tecla F4.


ㅤ+ Atalhos que facilitam:

Ctrl + Tab Desloca rapidamente de um arquivo para outro
Ctrl + T Seleciona todas células
Ctrl + G Abre a uma guia de busca dentro do seu arquivo
Ctrl + pgUp ou pgDown Sobe ou desce para o limite da planilha
Ctrl + B Salva o arquivo
Ctrl + N Adiciona negrito no texto selecionado
Alt + C Adiciona Bordas
Ctrl + X Recortar
Ctrl + Z Desfaz a última ação
Ctrl + Y Refaz a última ação
